## Deploying / Updating

Since this is a single-file site, updating is straightforward:

1. Edit `index.html` locally.
2. Commit and push to the `main` branch.
3. GitHub Pages picks up the change automatically — usually live within 60 seconds.

```bash
git add index.html
git commit -m "update: <brief description>"
git push origin main
```

> **Tip:** Go to **Settings → Pages** in this repo and confirm the source is set to `Deploy from branch → main → / (root)`.
